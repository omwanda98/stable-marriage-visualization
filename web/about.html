STABLE MATCHING MARRIAGE PROBLEM

STABLE MATCHING MARRIAGE PROBLEM is a real-time Stable Marriage Algorithm WEB APP Visualizer.

Using My STABLE MARRIAGE WEB VISUALIZER

You can edit the names of everyone in either group of male or female, you may also see their list by clicking on their arrow icon. Drag and drop to rearrange their preference list. You may also remove them with the trash icon, or add more with the add icon. Use the play button to start the visualization. Skip if you want to see immediately their best stable pairing.

Why the Technology {python and Javascript}

Websites require Javascript to enable complex user interactions such as drag and drop, input synchronization, and even controllable animations. If your browser has Javascript disabled, the visualizer cannot work at all. Python is for algorithm implementation as learned.
